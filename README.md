📝 Online To-Do List Web Application

📌 Project Description

The Online To-Do List is a simple and user-friendly web application that helps users manage their daily tasks efficiently. Users can add, complete, and delete tasks, making it easier to stay organized and productive.

🎯 Objective

The main objective of this project is to provide a simple platform for users to:

- Organize daily activities
- Track completed tasks
- Improve time management

⚙️ Technologies Used

- HTML – Structure of the webpage
- CSS – Styling and layout
- JavaScript – Functionality and interactivity
- Local Storage – To save tasks in the browser

✨ Features

- ➕ Add new tasks
- ✅ Mark tasks as completed
- ❌ Delete tasks
- 💾 Save tasks using local storage
- 📊 Display remaining task count

🛠️ How to Run the Project

1. Download or clone the repository
2. Open the folder
3. Double-click "index.html"
4. The project will open in your browser

📷 Output

The application contains:

- Input field to add tasks
- Task list section
- Completed task section
- Buttons for actions

🚀 Future Enhancements

- Add deadlines for tasks
- Add categories (Work, Personal, etc.)
- User login system
- Sync with database
